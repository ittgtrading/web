Deploy steps
1. Create a public repository named YOURUSERNAME.github.io
2. Upload index.html, services.html, contact.html, styles.css, and the assets folder
3. Settings, Pages, Deploy from branch, main, root
4. Open https://YOURUSERNAME.github.io

- Stock images are linked from Unsplash
- Replace with your own images if needed
- Logo appears at the top of every page
